# EventEule - Quick Start Guide 🦉

**The Most Important Steps in 5 Minutes**

---

## 1. Install Plugin (2 Minutes)

1. 📥 Download ZIP file from GitHub
2. 🔧 WordPress Admin → **Plugins** → **Add New** → **Upload Plugin**
3. 📁 Select ZIP file and install
4. ✅ Click **Activate**

**Done!** EventEule is now active.

---

## 2. Create First Event (2 Minutes)

### In the WordPress menu: **Events** 🎉

1. Click **Events** → **Add New**

2. **Enter title**
   - Example: "Summer Festival 2026"

3. **Fill in details** (right sidebar)
   - ✏️ **Start date**: 15.06.2026
   - ⏰ **Start time**: 14:00
   - 📍 **Location**: Huisheim Library
   - 🔗 **Registration link**: https://...

4. **Write description** (main area)
   - What happens at the event?
   - What should visitors know?

5. Click **Publish** (top right)

✅ **Your first event is live!**

---

## 3. Display Events on Website (1 Minute)

### Event List on a Page

1. **Pages** → **Add New**
2. Title: "Events"
3. Write in the editor:
   ```
   [eventeule_events]
   ```
4. **Publish**

✅ **Your event list is online!**

### Add to Navigation

1. **Appearance** → **Menus**
2. Find "Events" under "Pages"
3. Add to menu
4. **Save Menu**

---
